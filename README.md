# Rutgers Chemical Physics website

This is a static multi-page website prepared for GitHub Pages.

## Files
- `index.html` – homepage
- `about.html` – overview of the initiative
- `symposium.html` – symposium page with program summary and photo gallery
- `contact.html` – campus contacts
- `assets/` – styles, script, logo, and gallery images

## Deploy on GitHub Pages
1. Create a GitHub repository.
2. Upload all files from this folder to the repository root.
3. In GitHub, open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the main branch and the root folder (`/`).
6. Save. GitHub Pages will publish the site.

## Notes
- The site is plain HTML/CSS/JS, so it works directly on GitHub Pages.
- Update contact details or add email links in `contact.html` if desired.
- Replace or expand the gallery in `assets/images/gallery/` at any time.
